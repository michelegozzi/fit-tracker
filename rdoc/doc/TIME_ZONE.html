    Setting the default time zone for a Heroku app

    From the shell execute the following commands:

    $ heroku config:add TZ="Europe/Rome"
$ heroku config:add TZ="Europe/London"

    For more informations see:

    http://blog.pardner.com/2012/08/setting-the-default-time-zone-for-a-heroku-app/
http://en.wikipedia.org/wiki/List_of_tz_database_time_zones
